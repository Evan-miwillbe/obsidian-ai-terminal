## 1. 标签栏

```
┌──────────────────────────────────────────────────────┐
│ [Terminal 1 ✎ ×] [Git ✎ ×] [node ×] [+] [📄]       │
└──────────────────────────────────────────────────────┘
```

### 元素说明

| 元素 | 说明 |
|------|------|
| `Terminal 1` | 终端名称，默认 "Terminal N" |
| `✎` | 鼠标悬停时出现，点击弹出输入框重命名 |
| `×` | 点击弹出确认弹窗 "Close Terminal 1?"，确认后关闭 |
| `+` | 点击立即创建新终端 tab |
| `📄` | 点击复制当前活跃笔记的绝对路径到剪贴板 |

### 交互

- **点击标签** → 切换到该终端（终端内容保持，不清空）
- **拖拽标签** → 可拖到主面板底部创建分栏，拖出时有蓝色指示线
- **悬停** → 标签背景变亮，✎ 按钮出现
- **活跃标签** → 白色背景 + 灰色边框

## 3. 分割线

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  ← 可拖动
```

### 说明

- 5px 宽的灰色横条
- 鼠标悬停变为主题色
- 鼠标变为上下箭头（row-resize）
- 拖动时主面板和分栏高度实时调整

## 4. 分栏面板

```
┌──────────────────────────────────────────────────────┐
│ Git  [↑]                          [📄] [×]            │  Header
│ ───────────────────────────────────────────────────  │
│ PS D:\Knowledge-Vault> git status                    │
│ On branch main                                       │
│ nothing to commit, working tree clean                │
│ PS D:\Knowledge-Vault> █                             │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 元素说明

| 元素 | 说明 |
|------|------|
| `Git` | 终端名称，右键可重命名 |
| `↑` | 点击将分栏还原回标签栏 |
| `📄` | 复制当前笔记路径 |
| `×` | 关闭分栏并终止终端 |

### 交互

- 关闭分栏 → 该终端的 tab 重新出现在标签栏
- 多个分栏纵向堆叠，之间有细分割线
- 默认隐藏，拖入第一个标签后出现

## 5. 终端样式

### 终端内部

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│  PS D:\Knowledge-Vault> █                            │
│                                                      │
└──────────────────────────────────────────────────────┘
```

| 属性 | 值 |
|------|------|
| 字体 | Obsidian 等宽字体 |
| 字号 | 设置中可调 |
| 光标 | 紫色实心块，闪烁 |
| 背景 | 跟随 Obsidian 深色/浅色模式 |
| 圆角 | 6px |
| 内边距 | 上下 8px，左右 4px |

### 深色模式配色

| 元素 | 颜色 |
|------|------|
| 背景 | `#1e1f2eee` |
| 前景 | `#e2e4e9` |
| 光标 | Obsidian accent 色 |
| 选区背景 | `#264f78` |

### 浅色模式配色

| 元素 | 颜色 |
|------|------|
| 背景 | `#f5f5f5ee` |
| 前景 | `#1a1b1e` |
| 光标 | Obsidian accent 色 |
| 选区背景 | `#add6ff` |

## 6. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 插入换行 |
| `Ctrl+C`（有选区） | 复制选区到剪贴板 |
| `Ctrl+C`（无选区） | 发送 SIGINT（中断当前命令） |
| `Ctrl+V` | 粘贴剪贴板内容 |
| `Ctrl+滚轮上` | 放大字体 |
| `Ctrl+滚轮下` | 缩小字体 |
